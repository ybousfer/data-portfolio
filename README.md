# Data portfolio

Selected projects in analytics and data science: SQL, reporting, and data quality.



#### Projects



###### **Import/Export electronics analytics (SQL + data quality)**

Relational model redesigned as a business case study (shipments, lines, payments, clients, suppliers). Includes KPI queries and automated data quality checks (integrity, duplicates, date logic).

* Repo: https://github.com/ybousfer/electrotrade-import-export-case-study
* Stack: SQL (SQLite), Python, data modeling, data quality





###### **ML classification web app (FastAPI + React + Docker)**

End-to-end ML project: model training and evaluation, inference served via an API, and a small UI. Repository structured as a real application (backend/frontend/tests) with a reproducible setup.



\- Repo: https://github.com/ybousfer/02\_ml\_api\_app

\- Stack: Python, scikit-learn, FastAPI, React, Docker





###### **IMDb API ingestion pipeline**

Small data ingestion project built around an external API. The script retrieves movie information, extracts selected fields, and saves the result as a structured CSV file.



\- Repo: https://github.com/ybousfer/03\_data\_ingestion\_pipeline

\- Stack: Python, requests, pandas, API ingestion





###### **Migration research analysis**

Research-oriented analysis based on official Eurostat asylum application data. Includes data cleaning, transformation, and a small set of visual outputs focused on monthly evolution and country comparisons.



\- Repo: https://github.com/ybousfer/04\_migration\_research\_analysis

\- Stack: Python, pandas, matplotlib, official public data



#### Technical focus

* SQL analytics (joins, aggregation, KPI design)
* Data quality (integrity checks, anomaly detection, reproducible workflows)
* Python (automation scripts, notebooks)
